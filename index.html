How It Works

My part of the machine needs paired Arduino code for it to function properly. The input from the previous machine will start the process by actuating the gate at the top and allowing the cart to begin rolling down the ramp. This gate is connected to the Arduino circuitry, and it is implementing a short delay for the second gate. After this delay, the gate will open which allows for a small steel ball to drop into the cart. Afterwards, the cart will travel through the rest of the machine with the ball safely tucked inside. At the end of the last ramp, the cart will hit a lip which will eject the steel ball out of the cart and through the exit door at the end of the machine, becoming the input for the next part of the entire machine.
